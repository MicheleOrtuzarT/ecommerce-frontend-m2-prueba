# Ecommerce Frontend - Portafolio

## Descripción
Este proyecto es parte de mi portafolio del Módulo 2 del Bootcamp. Es una tienda virtual enfocada en el desarrollo de la interfaz (Frontend).


## Características
- Maquetación semántica con HTML5.
- Estilos personalizados con CSS3 y Bootstrap 5.
- Diseño Responsive (Adaptable a móviles).
- Interacciones con JS

## Tecnologías utilizadas
- HTML5 / CSS3
- Bootstrap 5
- Git / GitHub
- JS
